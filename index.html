introduction to windows 7

Windows 7 is the Microsoft Windows operating system (OS) launched in October 2009. Windows has simplified the task of managing data in a computer. The data in a computer is organized in the form of files and folders. In Windows, we can locate files and folders either in documents folder or Windows Explorer. It is a major release of the Windows NT operating system developed by Microsoft. It was released to manufacturing on July 22, 2009, and became generally available on October 22, 2009. It is the successor to Windows Vista, released nearly three years earlier. It remained an operating system for use on personal computers, including home and business desktops, laptops, tablet PCs and media center PCs, and itself was replaced in November 2012 by Windows 8, the name spanning more than three years of the product. Until April 9, 2013, Windows 7 RTM provided content such as security updates, software updates, PC driver updates and technical support, after which installation of Service Pack 1 is required for users to receive support and updates.







about windows explorer

The window of Windows Explorer is divided into two panes  - the left pane and the right pane. LEFT PANE - The left pane displays the disk drives and folders in a hierarchical order. RIGHT PANE – The right pane displays the contents of drive and folder that is selected in the left pane. There are 2 signs placed before the disk drive. The black trinagle and the white triangles are placed before the disk drive. The white triangle indicates that a folder contains subfolders.Clicking the white triangle will display the subfolders and it changes to a black triangle. This process is referred to as expanding.





To open Windows Explorer

Right click on the Start button and click on the Open Windows Explorer option OR Click on Start > All programs > Accessories > Windows Explorer
